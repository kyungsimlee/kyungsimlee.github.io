Project Idea

 

Overview

My project idea is to develop a smartphone application that can connect ‘food production services’ (such as café/restaurant/catering) to ‘customers’. Where customers can get involved in designing their food order according to needed nutrient intake or any specifications they may have. Customers can get involved in designing their food order from scratch starting from types of ingredients; methods of cooking; controlling sodium, fat and sugar content of the dish; to any unnatural chemicals added, for example hormone free meat; GMO free; MSG free etc.

 

Motivation

As ‘well-being’ living trends growing especially in food industry I think it will be worthwhile project to invest on as there is demand for target market. It is statically proven there is increasing consciousness of general public who are attentive to their daily nutrient intake. Also, as there is escalation of associated diseases of bad eating habits and having aged population requiring the need to control their daily nutrient consumption this smartphone  application can be a good contributor for solution.

 

 

Description

Functions of this smartphone application would include;  Customers are to design their food order from 1.) choosing ingredients (this can be types of ingredients to weather it is organic or not). 2.) adjusting fat/sugar/salt contents of the dish as well as any additional chemicals added for flavoring. 3.) preferred methods of cooking will be selected for the dish. (stir-frying, steaming etc.) 4.) delivery option is to be chosen. These are basic 4 simple steps that would be carried out to use this application. An additional feature is that what customers consumed as their food order can be collected as referential data for individual’s consumption on Nutrients. Furthermore, this can produce personalized reports in regular basis for recommended daily nutrient intake. Because this product requires first users to have background knowledge on food to design their order. A basic educating tutorial videos and texts will be provided for customers to utilize the application effectively. There will be templates offered for food orders with suggestive nutrient intake designed accordingly addressing various health issues for instance obesity and diabetes.One other additional feature will be having opportunities for complex linkage of different applications to improve life convenience. For example, this application can be connected to doctors and hospitals for referential information. This collected information can be utilized to develop another innovative application that can allow individuals to have a personalized nutritionist who can monitor and direct to achieve a balanced diet. 

 

Tools and Technologies

There is no requirement of hardware other than computer and mobile phone to create this product. Software that are needed for development would be different depending on whether this application is for iPhone or android phone. Tools required are as follows; Application development tools for iPhone: iOS specific technologies, like iOS SDK and others are used to develop applications for Apple mobile devices. iOS technologies are available with variety of development tools, like iOS stimulator, Objective C, Xcode etc. JavaScript, CSS and HTML are frequently used programming languages for building and deploying web-based iPhone applications. Application development tools for Android: Development for Android application rely on Android SDK. The Android specific development toolkit is primarily composed of two important components. One is Android Development Tools Plugin for Eclipse and the other one is Android emulator.

 

 

Skills Required

Referring to Tools and Technologies from above, basic compulsory skill required is Programming skills (coding). Such as C#, Java and trained in web development languages like HTML 5 and CSS. And as well as familiarized with mobile platform application programming interfaces (API) for example Apple iOS, Android. Secondly, important aspect of mobile app development would be building a high-quality user interface (UI). This is simply smooth interaction between the user and the software itself. Good practical experience in programming will be needed to achieve this. Overall it is feasible to find skills, software and hardware to launch this project.

 

 

 

 

Outcome

If my project is successful, the outcome will offer more options to general public on healthy eating. This will promote good eating habits as it is easy to plan and have balanced diet via using this application. Individuals with less to serious health issues will benefit by having more affordable personalized meal planner for their specific dietary needs. Furthermore, this app will open opportunities for complex linkage of different apps to improve life convenience. For example, this application can be connected to doctors and hospitals for referential information.
